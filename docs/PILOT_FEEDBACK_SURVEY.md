# Pilot Feedback Survey

Please answer the following after the guided session.

1. Overall, how would you rate the Investigation Canvas? (1-5)
2. How easy was it to find relevant events and link them in the timeline? (1-5)
3. Did the annotation workflow meet your needs? (Yes/No) If no, please explain.
4. Which features were missing or confusing?
5. How much time (estimate) did this workflow save compared to your usual process? (minutes)
6. Would you recommend this tool to a colleague? (Yes/No)
7. Any other comments or suggestions?

Please file any major blockers as GitHub issues and tag @maintainers.
