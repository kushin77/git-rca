# Roadmap (High-Level)

This roadmap maps epics into a set of milestones for an 6-month MVP timeline. Adjust timing after stakeholder alignment.

Month 0-1: Discovery & Governance
- Finalize product charter, KPIs, stakeholder interviews, pilot customer identification.

Month 2-3: Core Platform & Data Foundations
- Scaffold monorepo and CI/CD.
- Implement basic IaC for dev environment.
- Build 2 initial connectors (git events, CI) and load sample data.

Month 4: RCA Workflow MVP
- Basic investigation workspace UI.
- Timeline visualization and simple annotations.
- Exportable RCA report.

Month 5: Observability & Security
- Dashboards for ingestion and platform health.
- Threat model and RBAC baseline.

Month 6: Pilot & Adoption
- Onboard pilot teams, iterate based on feedback, finalize MVP scope for public beta.
